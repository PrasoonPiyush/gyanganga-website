# Contributing to GyanGanga

Thank you for considering contributing to **GyanGanga**!  
We welcome improvements, bug fixes, and new ideas.

## How to Contribute
1. Fork the repo: [prasoonpiyush/gyanganga-website](https://github.com/prasoonpiyush/gyanganga-website)
2. Create a branch: `git checkout -b feature/your-feature`
3. Make changes in `docs/`
4. Commit: `git commit -m "Add feature"`
5. Push: `git push origin feature/your-feature`
6. Open a Pull Request

## Guidelines
- Follow accessibility best practices
- Keep code clean and documented
- Use descriptive commit messages
- Respect the [Code of Conduct](CODE_OF_CONDUCT.md)

## Issue Reporting
- Use the provided templates for bug reports and feature requests
- Include screenshots or logs when possible
